# Система кадрового учёта

Добро пожаловать в документацию по веб-приложению для автоматизации кадрового учёта.

## О проекте

Веб-приложение, в котором специалист по кадрам ведет учет сотрудников в нескольких организациях. Позволяет вести учёт организаций, отделов, должностей, сотрудников, кадровых операций, загружать сканы паспортов, отслеживать историю изменений и управлять доступом пользователей с разными ролями. Разработано в рамках производственной практики.

## Основные возможности

- **Управление организациями** – создание, редактирование, мягкое удаление.
- **Отделы** – вложенная структура, привязка к организациям.
- **Должности** – справочник.
- **Сотрудники** – личные данные, паспорт, адрес, загрузка сканов.
- **Кадровые операции** – приём, изменение зарплаты, изменение отдела, увольнение.
- **История изменений** – автоматическое логирование всех действий (кто, когда, что изменил).
- **Аутентификация и роли** – вход по логину/паролю (Argon2), сессии, роли «Администратор» и «Менеджер по персоналу».
- **Защита маршрутов** – доступ к разделам в зависимости от роли.

## Технологический стек

- **Frontend**: Vue 3, Vite, Pinia, Axios
- **Backend**: Node.js (Express), Passport.js, Argon2, Joi, node-pg-migrate
- **База данных**: PostgreSQL 17
- **Контейнеризация**: Docker, Docker Compose
- **Документация**: VitePress
