# Кадровые операции

Позволяют фиксировать:

- Приём на работу,
- Изменение зарплаты,
- Изменение отдела,
- Увольнение.

## Просмотр операций

Таблица показывает:

- ID – уникальный номер,
- Сотрудника,
- Тип операции,
- Отдел, должность, зарплату (в зависимости от типа),
- Дату создания.

## Добавление операции

1. Нажмите **«Добавить операцию»**.
2. Выберите **сотрудника**.
3. Выберите **тип операции**:
   - **Принятие на работу** – обязательны отдел, должность, зарплата.
   - **Изменение зарплаты** – обязательна новая зарплата.
   - **Перевод в другой отдел** – обязателен новый отдел.
   - **Увольнение** – дополнительные поля не нужны.
4. Заполните необходимые поля и нажмите **«Сохранить»**.

## Удаление

- **«Удалить»** – мягкое удаление записи об операции.
