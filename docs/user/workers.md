# Сотрудники

Здесь хранятся личные данные, паспортная информация, адрес регистрации и прикреплённые скан-копии паспорта.

## Просмотр списка

Таблица показывает:

- ID,
- ФИО,
- дату рождения,
- серию и номер паспорта,
- краткий адрес (населённый пункт, улица, дом).

## Добавление сотрудника

1. Нажмите **«Добавить сотрудника»**.
2. Заполните **Личные данные** (фамилия, имя, отчество, дата рождения).
3. Заполните **Паспортные данные** (серия, номер, дата выдачи, код подразделения, кем выдан).
4. Заполните **Адрес** (регион, населённый пункт, улица, дом, корпус, квартира).
5. Нажмите **«Сохранить»**.

## Редактирование

Кнопка **«Ред.»** открывает форму, где можно изменить любые данные.

## Файлы (сканы паспорта)

- Нажмите **«Файлы»** у нужного сотрудника.
- Вы можете загрузить несколько файлов (заполнить название и выбрать файл).
- Загруженные файлы можно скачать или удалить.

## Удаление сотрудника

Кнопка **«Удалить»** помечает сотрудника как удалённого (мягкое удаление). Все связанные кадровые операции и файлы сохраняются.
