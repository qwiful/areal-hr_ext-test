# Руководство пользователя

Это приложение предназначено для специалистов по кадрам и администраторов. Оно позволяет вести учёт организаций, отделов, должностей, сотрудников, кадровых операций, а также отслеживать историю изменений.

## Как пользоваться руководством

В левом меню выберите интересующий раздел. Каждый раздел содержит:

- описание возможностей,
- пошаговые инструкции.

## Требования к браузеру

Рекомендуется использовать последнюю версию Google Chrome.
