# Вход в систему

Для доступа к приложению необходимо ввести логин и пароль, выданные администратором.

## Форма входа

1. Откройте приложение.
2. Введите **логин** и **пароль**.
3. Нажмите кнопку **«Войти»**.

## Возможные ошибки

- **«Неверный логин или пароль»** – проверьте правильность ввода. Убедитесь, что Caps Lock выключен.
- **«Необходима авторизация»** – сессия истекла, войдите снова.

## Выход из системы

Нажмите кнопку **«Выйти»** в боковом меню.
