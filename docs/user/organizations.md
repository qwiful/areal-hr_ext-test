# Организации

В этом разделе вы можете управлять списком организаций, в которых ведётся кадровый учёт.

## Просмотр списка

Таблица показывает все организации, которые не были удалены. Для каждой организации отображается:

- **ID** – уникальный номер,
- **Название**,
- **Комментарий**.

## Добавление организации

1. Нажмите кнопку **«Добавить организацию»**.
2. Заполните поля:
   - **Название** (обязательно, до 50 символов)
   - **Комментарий** (необязательно, до 250 символов)
3. Нажмите **«Сохранить»**.

## Редактирование и удаление

- Кнопка **«Редактировать»** позволяет изменить любые поля.

- Кнопка **«Удалить»** выполняет мягкое удаление – запись помечается как удалённая, но остаётся в базе данных. В списке она больше не отображается.
