# Отделы

Отделы могут быть вложенными (например, «Коммерческий департамент» главный для «Отдел продаж»). Каждый отдел обязательно принадлежит какой-либо организации.

## Просмотр списка

Таблица показывает:

- **ID** – уникальный номер,
- Название отдела,
- Организацию,
- Родительский отдел (если есть),
- Комментарий.

## Добавление отдела

1. Нажмите **«Добавить отдел»**.
2. Введите **название** (обязательно).
3. Выберите **организацию** из выпадающего списка.
4. При необходимости выберите **родительский отдел** (чтобы создать вложенную структуру).
5. Добавьте **комментарий** (необязательно).
6. Нажмите **«Сохранить»**.

## Редактирование и удаление

- Кнопка **«Редактировать»** позволяет изменить любые поля.
- Кнопка **«Удалить»** – мягкое удаление (отдел пропадает из списка, но данные сохраняются в базе).
