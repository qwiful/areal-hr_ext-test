# История изменений

Автоматически фиксирует все действия пользователей: создание, редактирование, удаление любых объектов (организаций, отделов, сотрудников и т.д.).

## Просмотр

- Выберите **фильтр по объекту** (например, «Организация», «Сотрудник»), чтобы отобразить только нужные изменения.
- Таблица показывает:
  - ID – уникальный номер,
  - Дату и время операции,
  - Кто изменил (ФИО специалиста),
  - Объект,
  - Изменённые данные (в формате JSON – можно развернуть, нажав «Показать»).
- Навигация по страницам позволяет просматривать записи по 20 штук.

## Что считается изменением?

- Создание новой записи.
- Обновление любой информации.
- Мягкое удаление (пометка `delete_at`).

## Редактирование и удаление

Историю нельзя редактировать или удалять - она служит для аудита.
