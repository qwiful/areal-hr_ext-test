# Должности

Справочник должностей, которые могут быть назначены сотрудникам.

## Просмотр

Таблица показывает:

- ID – уникальный номер,
- Название должности.

## Добавление

Нажмите **«Добавить должность»**, введите название и сохраните.

## Редактирование и удаление

- **«Редактировать»** – изменить название.
- **«Удалить»** – мягкое удаление (должность перестаёт отображаться, но остаётся в базе).
