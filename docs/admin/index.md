# Руководство администратора

Данный раздел предназначен для специалистов, которые устанавливают, настраивают и сопровождают систему кадрового учёта.

## Что вы найдёте здесь

- **Установка и запуск** – как развернуть приложение на сервере или локально.
- **Конфигурация** – переменные окружения, настройки CORS, сессий.
- **Запуск через Docker** – контейнеризация всего стека (PostgreSQL, backend, frontend, nginx).
- **База данных и миграции** – управление схемой БД, ручной запуск миграций.
- **Пользователи и роли** – создание администратора или менеджера.

## Требования к серверу

- **Операционная система**: Linux (Ubuntu 20.04+), macOS, Windows с WSL2.
- **Node.js**: версия 22 или выше.
- **PostgreSQL**: версия 17 или выше (или Docker-контейнер).
- **Docker** (опционально): версия 20.10+ и Docker Compose v2.

## Архитектура приложения

- **Frontend**: Vue 3, раздаётся через nginx.
- **Backend**: Node.js + Express, REST API на порту 3000.
- **База данных**: PostgreSQL, таблицы создаются через миграции.
- **Прокси**: nginx перенаправляет запросы `/api/*` на backend.
